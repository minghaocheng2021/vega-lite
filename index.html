Choropleth Map

To start with the analysis, I first plotted a choropleth map showing the mean price of houses in different suburbs of Melbourne. A choropleth map is a thematic map in which different regions in the map shaded or patterned in proportion to a statistical variable that represents an aggregate summary of a geographic characteristic. (Wikipedia)
